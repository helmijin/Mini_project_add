# 🖥 Mini_project_add 소개
최신 영화에 대한 간단한 정보 제공으로, 문화 사업을 활성화하고 문화적 소양을 끌어 올리기 위해 선정 및 제작  
최신 영화에 대한 정보를 확인할 수 있으며, "My List"기능 추가로 원하는 영화를 찜할 수 있음
